# 批量版变体

> 多篇文献合并分析，每篇作为独立章节。

## 提示词

我上传了多篇PDF文献，请依次对每篇进行「逐段对照深度解析」，最终合并为一个Word文档报告。

对每篇文献执行以下流程：

【第一步】提取全文，识别文献结构。

【第二步】对摘要的每个段落，生成逐段对照表（原文 + 中文翻译 + 术语解释）。

【第三步】对每张Figure的每个小图生成解析表（小图编号 | 实验内容 | 结果解读），每张图后附「关键结论」。

【第四步】方法总结表格（制备方法、动物模型、检测方法）。

【第五步】结论与意义（主要结论、临床转化意义、研究局限）。

【第六步】术语表（10-15个核心术语）。

【合并要求】
- 每篇文献作为独立章节，使用H1标题分隔
- 章节标题格式：「第X篇：文献标题」
- 文档开头生成总目录页，列出所有文献标题和页码
- 文档末尾生成对比总结表，横向对比所有文献的：
  - 研究主题
  - 核心方法
  - 主要发现
  - 创新点

【输出格式】
- 生成单个.docx文件
- 蓝色表头（#2B6CB0），微软雅黑+Arial字体
- 关键结论红色标注（#C53030）

BATCH_MODE: true
ANALYSIS_SCOPE: abstract+results+discussion
FIGURE_DEPTH: detailed
