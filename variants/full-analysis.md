# Full Analysis Variant

> Complete analysis of all sections. ~15 min per paper. Default mode.

## Prompt

请对这篇文献进行「完整逐段对照深度解析」，生成Word文档报告。严格按照以下流程执行：

【第一步：提取全文】
使用PDF技能提取文献全部文本内容（分页提取），识别文献结构（摘要、引言、方法、结果、讨论、结论）。

【第二步：逐段对照解析】
对摘要和引言的每个段落，生成三栏对照表：
- 左栏：英文原文
- 右栏：中文翻译（专业准确，术语统一）
- 底部：术语解释（提取该段中的专业术语，给出中英文对照和简明定义）

对结果和讨论部分，分析包含主要发现、机制解释、统计结果的关键段落。

【第三步：图表深度解析】
对文章中每张组图（Figure/Scheme），为每个小图（a, b, c...）生成三栏解析表：
- 第1栏：小图编号
- 第2栏：实验内容描述（该小图做了什么实验、用了什么方法、检测了什么指标）
- 第3栏：结果解读（数据说明了什么、与对照组相比的差异、统计学意义）
每张图解析完后，附一段「关键结论」总结该图的核心发现。

【第四步：方法总结】
用表格形式汇总：
- 纳米材料/药物的制备方法和关键参数
- 动物模型信息（种类、诱导方式、分组方案）
- 主要检测方法（检测类型 → 方法 → 检测指标）

【第五步：结论与意义】
- 主要结论（5条以内）
- 临床转化意义
- 研究局限与未来方向

【第六步：术语表】
用表格列出文中所有核心术语，包含：英文术语、中文翻译、简明定义（10-15个）。

【输出格式要求】
- 生成.docx文件，保存到工作文件夹
- 文档标题：文献深度解析报告
- 使用专业排版：封面页、分章节标题、表格带表头颜色
- 中文字体使用微软雅黑，英文字体使用Arial
- 所有表格使用蓝色表头（#2B6CB0）
- 关键结论用红色（#C53030）标注

ANALYSIS_SCOPE: all
FIGURE_DEPTH: detailed
TRANSLATION_LANG: chinese
