# 文献深度解析技能

## 概述

本技能用于对科学文献进行逐段对照深度解析，生成包含中英双语对照、图表深度解读、方法总结、术语表的专业Word文档。

## 调用方式

**必须显式调用**，例如：
- `用 literature-analysis 解析这篇文献`
- `使用 literature-analysis 技能分析这个文件夹中的论文`
- `$literature-analysis 解析所有PDF`

## 支持的输入

| 输入类型 | 示例 |
|---------|------|
| 单篇PDF | 上传一个PDF文件 |
| 多篇PDF | 上传多个PDF文件 |
| 本地文件夹 | `/Users/xxx/papers/` 包含多篇PDF |

## 工作流程

### 第一步：识别输入类型

- **单篇PDF**：直接解析，生成单个报告
- **多篇PDF**：依次解析，合并为一个报告（每篇作为独立章节）
- **本地文件夹**：扫描文件夹中所有 `.pdf` 文件，批量解析

### 第二步：全文提取

使用 PDF 技能提取论文的全部文本内容：
- 使用 `extract_pages.py` 逐页提取
- 识别文档结构：标题、作者、摘要、引言、方法、结果、讨论、结论
- 记录所有 Figure 和 Scheme 的编号及图注

### 第三步：逐段对照解析

对**摘要**和**引言**部分，为每个段落创建三栏对照表：

| 栏位 | 内容 |
|------|------|
| 原文 | 论文中的英文原文 |
| 翻译 | 准确的中文翻译，术语统一 |
| 术语解释 | 提取该段落中的专业术语，提供中英对照和简明定义 |

**术语格式**：`• 英文术语（中文翻译）：简明中文定义`

### 第四步：图表深度解析

对论文中的**每张 Figure 和 Scheme**，为**每个小图**（a, b, c, d...）创建详细解析表：

| 栏位 | 内容 |
|------|------|
| 小图编号 | 标签（如 "a", "b", "c"） |
| 实验内容 | 做了什么实验、用了什么方法、检测了什么指标 |
| 结果解读 | 数据说明了什么、与对照组的差异、统计学意义 |

分析完一张图的所有小图后，添加一段**关键结论**（用红色标注），总结该图的核心发现。

### 第五步：方法总结

创建结构化汇总表：

**表1：材料/药物制备**
| 参数 | 详情 |
|------|------|
| 组分 | 列出所有组分及比例 |
| 方法 | 制备技术 |
| 关键参数 | 粒径、PDI、包封率等 |

**表2：动物模型**
| 参数 | 详情 |
|------|------|
| 物种/品系 | 如 DBA/1小鼠、C57BL/6 |
| 模型 | 如 CIA、肿瘤移植 |
| 诱导方式 | 如何建立模型 |
| 分组 | 治疗组及样本量 |
| 给药 | 剂量、途径、频率、时长 |

**表3：检测方法**
| 检测类型 | 方法 | 检测指标 |
|----------|------|----------|

### 第六步：结论与意义

**6.1 主要结论**（最多5条，每条应有数据支撑）

**6.2 临床转化意义**

**6.3 局限性与未来方向**

### 第七步：核心术语表

创建10-15个核心术语的表格：

| 英文术语 | 中文翻译 | 定义 |
|----------|----------|------|

## 批量处理模式

当输入为多篇PDF或文件夹时：

1. **依次解析**：每篇文献独立解析
2. **合并报告**：生成单个Word文档，每篇作为独立章节
3. **对比总结**：文档末尾添加横向对比表：

| 文献 | 研究主题 | 核心方法 | 主要发现 | 创新点 |
|------|---------|---------|---------|--------|
| 文献1 | ... | ... | ... | ... |
| 文献2 | ... | ... | ... | ... |

## 输出格式要求

生成 .docx 文件，具体要求：

1. **封面页**：论文标题（中英双语）、期刊名称、发表日期、作者、机构
2. **章节标题**：H1 用于主要章节，H2 用于子章节，H3 用于单个段落/图表
3. **表格**：蓝色表头（#2B6CB0），白色文字，交替行颜色
4. **字体**：中文微软雅黑，英文 Arial
5. **关键结论**：用红色（#C53030）标注
6. **页码**：页脚居中
7. **页眉**：论文标题缩写，右对齐

## 使用示例

### 单篇PDF
```
用 literature-analysis 解析这篇文献
```

### 多篇PDF
```
用 literature-analysis 解析这些文献，合并为一个报告
```

### 本地文件夹
```
用 literature-analysis 解析 /Users/xxx/papers/ 文件夹中的所有PDF
```

### 快速模式
```
用 literature-analysis 快速解析这篇文献（只看摘要和图表）
```

## 配置选项

可在请求中追加：

```
ANALYSIS_SCOPE: abstract+results+discussion  # 只解析这三个部分
FIGURE_DEPTH: brief                          # 每个小图1-2句话
OUTPUT_SEPARATE: true                        # 多篇文献分别生成独立报告
```
